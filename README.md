# javascript2

![Project Logo](img/rewind-logo.png)

JavaScript techniques applied to implement the front-end functionality for a social media application.
This is a social media platform aiming for a retro vibe, both in the choice of name and in the design.
It is meant to be a way to share photos with friends and family online.

## Description

User with @noroff.no or @stud.noroff.no email can register profile

Social media site allowing a logged in user:

- view/filter posts
- view a post content item by ID
- create a post
- update a post
- delete a post
- search posts feed

## Features

- Dark/light mode
- Responsive design

## Live site

[Rewind.](https://rewind-rewind.netlify.app/)

## Setup

```bash
npm install
```

For development:

```bash
npm run dev
```

For production

```bash
npm run build
```

## Tech stack

- HTML
- Vanilla JS
- Tailwind CSS
